# Claude Opus 4.8 性能评估与实测简报

## 执行摘要

Anthropic 发布了全新模型 Claude Opus 4.8，标志着该公司重回全球 AI 性能巅峰。随同新模型发布，Anthropic 宣布完成 650 亿美元融资，估值达到 9650 亿美元，一举超越了 OpenAI 的估值水平。Claude Opus 4.8 在编程架构思考、拟人化写作及企业级数据处理方面表现卓越。尽管其核心优势依赖于高强度的“Ultra High”推理模式，且在特定细分测试及交互体验上仍存争议，但该模型在透明度与综合实力上的提升使其成为当前复杂工程项目的首选工具。

## 核心要点

Claude Opus 4.8 在编程能力上实现了跨越式进步，解决了前代版本 4.7 在高级工程基准测试中的短板。通过开启“Ultra High”高强度推理模式，其编程评分达到 63 分，比 4.7 版本提升 30 分，并以微弱优势超越 [[GPT-5.5]]（62 分）。该模型能够处理真实的生产级代码库，展现出架构化思考能力，从“代码补全工具”进化为具备工程思维的“AI 工程师”。在拟人化写作方面，Claude Opus 4.8 在综合评测中获得 79.6 分，位居行业第一。用户只需提供风格模板，模型即可精准模仿语气，摆脱了常见的 AI “机械腔”。面对深层心理问题时，模型会质疑前提假设，展现出真实的思考深度。在企业与学术端，该模型表现同样亮眼：企业报告提取准确率达 87%，法律合同审查接近完美，财务分析能力提升约 8 个百分点。在沃顿商学院的测试中，模型能完成数据清洗、假设提出、文献调研及论文撰写的全流程任务，还可主动修正其他 AI 模型（如 GPT-5）的审稿幻觉。然而，Claude Opus 4.8 的神级表现高度依赖“Ultra High”模式，一旦降档至普通模式，编程得分暴跌至 40 分，写作风格回落至平庸的“AI 套路腔”。该模式消耗极快，用户频繁触碰每月 200 美元的额度上限。此外，在终端编程测试中，Claude Opus 4.8 以 74.6 分落后于 [[GPT-5.5]] 的 78.2 分。实测显示，前端开发的设计美感与细节逻辑优于前任，科学模拟（太阳系动画、万有引力沙盒）效果突出，但流体力学逊色于 [[Google]] 相关模型，多米诺骨牌动态物理效果不够细腻。视觉识别测试中，模型能正确识别筷子数量（19 根）。总体而言，Claude Opus 4.8 最适合需要高强度推理的重度用户，普通用户需权衡成本与交互体验，中国大陆用户还面临严格的封号风险。后续 Anthropic 已预告更强的“MOS 模型”，竞争格局将继续剧变。
